# Install

Create a virtual environment:

    $ pyenv virtualenv 3.6.4 wosmongo

Install all the requirements:

    $ pip install -r requirements.txt

Open the Jupyter notebooks:

    $ jupyter notebook

Or run the example:

    $ cd scripts
    $ python example1.py
